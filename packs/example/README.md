# Example pack workspace

This directory is intentionally kept as an empty placeholder. It is not a
loadable or shipped art pack: it has no `pack.json` and contains no legacy
placeholder assets.

The example pack and its Art Kit will be rebuilt from the final v4 visual and
asset specifications after v4 is locked. Until then, use
[`docs/extending.md`](../../docs/extending.md) and
[`docs/assets.md`](../../docs/assets.md) for current engine constraints only.
