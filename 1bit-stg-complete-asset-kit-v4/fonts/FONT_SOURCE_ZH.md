# 字体来源与许可

## 使用字体

- 名称：Noto Sans SC Variable
- 上游：Google Fonts 官方仓库 `google/fonts`
- 字体文件：`ofl/notosanssc/NotoSansSC[wght].ttf`
- 来源：https://github.com/google/fonts/tree/main/ofl/notosanssc
- 字体原始文件：https://raw.githubusercontent.com/google/fonts/main/ofl/notosanssc/NotoSansSC%5Bwght%5D.ttf
- 许可原文：https://github.com/google/fonts/blob/main/ofl/notosanssc/OFL.txt
- 许可：SIL Open Font License 1.1
- 核实日期：2026-07-17

本目录中的 `OFL.txt` 是上游同目录许可文件的原样副本。字体未改名、未改造、未子集化，仅由 Pillow 用于生成 mockup 中的中英文字。

## 文件校验

- `NotoSansSC-Variable.ttf` SHA-256：`a3041811a78c361b1de50f953c805e0244951c21c5bd412f7232ef0d899af0da`
- `OFL.txt` SHA-256：`1c05c68c34f9708415aada51f17e1b0092d2cea709bf4a94cd38114f9e73d7d9`

发布游戏时可继续随包分发该字体与许可文件，也可以只把已经栅格化的字形放入自有位图字体图集；两种情况都应保留 `OFL.txt` 与来源记录。
