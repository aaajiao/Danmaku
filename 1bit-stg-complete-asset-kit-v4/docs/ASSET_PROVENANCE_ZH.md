# 素材来源与生产说明

## 像素图

V4 三张新增源板由本任务中的图像生成系统依据《1bit》世界观、aaajiao 方法论与既有 V3 像素合同生成；随后通过包内脚本进行确定性处理：绿色键控去底、八色量化、单元裁切、四可见色限制、硬透明和安全边距检查。最终运行时图集不是未经处理的生成图。

生成源板保存在 `sources/generated/`，复建脚本保存在 `sources/scripts/`。使用最终图集时以 `manifests/v4/frame-index-v4.json` 为准。

## 音频

48 个 WAV 由 `audio/generate_audio_v4.py` 程序化合成。生成器、采样率、响度、循环信息和 SHA-256 均记录在 audio manifest；重复生成哈希一致。

## 字体

UI 使用 Noto Sans SC。字体授权为 SIL Open Font License 1.1，许可证随包位于 `fonts/OFL.txt`。

## 历史素材

四张基础图集、基础背景与 UI atlas 来自同一项目的 V3 资源包，在 V4 中作为兼容基础继续使用。历史文档和旧预览集中于 `legacy-v3/`，不构成 V4 权威运行时逻辑。

## 使用边界

本包面向当前《1bit》项目制作。若要对外发布、转授权或用于其他产品，应由项目权利人按自身发布策略确认最终授权文本；本说明记录生产来源，不替代法律许可文件。
